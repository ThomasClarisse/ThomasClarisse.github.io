Projects

project 1: calculator

This is a project i made using php, html and css, it is a simple calculator running in the web browser. The calculator can do basic operations such as addition, subtraction, multiplication and division. It can also give an error message when you try to divide by 0 or enter a letter instead of a number.





project 2: rock paper scissors

This is a project that me and a classmade had to make for our group project. It's a simple rock paper scissors game made with php, html and css. You can choose wether you play against another player (on the same device) or against the computer.





Project 3: Hangman

This project is also the result of a group exercise with me and my classmate, we had to make a fully working hangman game sing php, html and css The game has the option to choose your own word or have the computer choose a random word from a predifined array. You have a limited amount of tries to guess the word, if you fail you lose the game.





Project 4: sql database

This project is a series of databases made in mySQL. The first database is a simple database containing the planets in our solar system and their respective data. The second database is slightly more complex as it is for a fictional steaming service called "netland", it contains tables for movies and their data. The third database is for the same streaming service but this time it contains tables for shows and all their data.





Project 5: netland admin panel

In this project i had to make a admin panel for the previously mentioned netland database. The admin panel has a option to sort the movies and shows by their title (a-z and z-a) or by their rating (highest to lowest and lowest to highest). You can also add new movies and shows to the database directly from the admin panel, editing the allready existing entries is also possible once you click on a title to show it's details. The entire admin panel is only accessible after logging in with the correct credentials. The admin panel is made using php, html, css and pdo for the database connection.
